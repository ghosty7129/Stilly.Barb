# 🚀 Quick Start Guide

## Prerequisites

Before running this project, make sure you have Node.js installed:

1. Download and install Node.js (v18 or higher): https://nodejs.org/
2. Verify installation by opening a terminal and running:
   ```
   node --version
   npm --version
   ```

## Installation & Setup

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Start Development Server**

   ```bash
   npm run dev
   ```

   The site will open at `http://localhost:3000`

3. **Build for Production**
   ```bash
   npm run build
   ```

## First Time Setup

After installation, your project will be ready to run. The development server includes:

- ✅ Hot Module Replacement (instant updates)
- ✅ All dependencies installed
- ✅ Tailwind CSS configured
- ✅ React Router set up
- ✅ Framer Motion animations ready
